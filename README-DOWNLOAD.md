# How to Share the Website with Your Client

## Quick Options:

### Option 1: WeTransfer (Easiest - 7 days)
1. Go to https://wetransfer.com
2. Upload `index.html`
3. Enter your email and client's email
4. Send - they'll get a download link

### Option 2: Google Drive
1. Upload `index.html` to Google Drive
2. Right-click → Share → Get link
3. Change permission to "Anyone with the link"
4. Copy link and share

### Option 3: Netlify Drop (Live Preview + Download)
1. Go to https://app.netlify.com/drop
2. Drag entire folder or just `index.html`
3. Get instant live URL (e.g., `https://xyz-123.netlify.app`)
4. Client can view AND download

### Option 4: Direct Download Page
- Open `download.html` in browser
- Click download button
- Or share the download.html page

## File to Share:
- **index.html** - The complete website (70KB)

## Notes:
- The website uses external resources (Tailwind CDN, Unsplash images)
- Works offline once downloaded
- No server needed - just open in browser
